# employee-sql-analysis

This project demonstrates SQL concepts from **basic to advanced level**
using an employee database.

## Features
- Table creation with constraints
- Constraint modification using ALTER
- Realistic employee dataset (50+ records)
- Easy, Medium & Advanced SQL queries
- CASE statements, GROUP BY, HAVING, BETWEEN
- Duplicate detection & salary analysis

## Database
- PostgreSQL

## Topics Covered
- DDL (CREATE, ALTER)
- DML (INSERT, DELETE)
- Aggregations
- Filtering
- Sorting
- Conditional logic (CASE)
